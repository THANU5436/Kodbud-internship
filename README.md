# Spam Email Classifier

## Overview
This project is a Machine Learning based Spam Email Classifier that detects whether an email/message is Spam or Ham (Not Spam).

## Technologies Used
 Python
 Pandas
 Scikit-learn
 TF-IDF Vectorizer
 Naive Bayes Algorithm

## Features
  Text preprocessing
  TF-IDF feature extraction
  Spam/Ham prediction
  Simple and fast model

## Installation

Install required libraries:

pip install pandas scikit-learn

## How to Run

python spam_classifier.py

## Working
1. Load dataset
2. Convert text into vectors using TF-IDF
3. Train Naive Bayes model
4. Predict whether message is Spam or Ham

## Example Output

Prediction: spam

## Applications
  Email filtering
  SMS spam detection
  Fraud message detection
